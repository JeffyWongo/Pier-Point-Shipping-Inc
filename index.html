9) We would like to point out the following limitations:

  • We did not implement "recovery from power outage".
  • We did not implement buffer
  • The program only prompts login scren once
  • The estimated time for load does not account for moving the crane to the initial position
  • [Balance]
    • We did not implement "total time estimate"
    • When balacning, tkinter prints a warning
  • [Load / Unload]
    • We did not implement "current time estimate"
